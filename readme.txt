Set No Sleep Mode for IOS and Your PWA using Wordpress Widgets.

== Description ==
Prevent your display from going to sleep on mobile devices by using a widget anywhere on your website.
it's also useful when you are developing PWA application for ios devices.
this plugin uses nosleep.js from repository below :
https://github.com/richtr/NoSleep.js

if you want your widget to be invisible you can delete the widget title .
